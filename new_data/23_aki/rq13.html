DMA(Direct Memory Access)は、コンピュータと周辺機器とデータのやり取りを制御する伝送路のことで、Direct Memory Accessの名の通り、入出力装置がCPUを介さずにメモリとの間で直接データを転送する方式です。
CPUが入出力装置の制御を行わなくてもよいため、データ転送中も別の処理を行うことが可能になります。
